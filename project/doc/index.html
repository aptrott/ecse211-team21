DPM BrickPi Programming Packages

 
Useful Modules
        
utils.brick

DPM Helper Code: Provides access to BrickPi sensors, motor, and Brick

 utils.sound

DPM Helper Code: Provides methods to create sound from attached speakers

 brickpi3

brickpi3 interfacing library

 simpleaudio

Sound producing library. Plays wave files and byte arrays
utils.filters

DPM Helper Code: Provides filters for numerical data (uses lists as input)

 utils.telemetry

DPM Helper Code: Provides simple GUI access for displaying information

 utils.remote

DPM Helper Code: Provides access to remote
(over the network) Brick capabilities

 utils.dummy

DPM Helper Code: Provides access to simulated Brick
hardware (sensors/motors) that can be used on any Computer
utils.rmi

(Advanced Code): The core tool used by utils.remote,
to create(client-side) and register(server-side) Remote Objects
